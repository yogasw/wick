# template

Agent-first Go service scaffold.

## Quick start
```
make setup
make dev
```

Open:
- Tool: http://localhost:8080/tools/convert-text
- Job (operator): http://localhost:8080/jobs/auto-get-data
- Job (admin): http://localhost:8080/manager/jobs/auto-get-data
